# 📌 Overview

HireLens AI is a modern AI-powered Career Intelligence Platform built to help students and fresh graduates prepare for internships and placements.

The platform combines resume analysis, ATS compatibility checking, skill gap detection, career readiness tracking, AI-powered career coaching, and job match prediction into one intuitive dashboard.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/tanug0/HireLens-AI.git
```

Move into the project folder

```bash
cd HireLens-AI
```

Install dependencies

```bash
npm install
```

Run development server

```bash
npm run dev
```

Create production build

```bash
npm run build
```
